Programme qui créé ou modifie des playlists qui devraient être lues
de façon aléatoire non décrémentale, auquel cas ce générateur permet
d'ajouter une probabilité d'être joué à chaque fichier, qui dans la
playlist de sortie s'exprimera par la répétition de l'entrée pour ce
fichier.
Le programme peut lire des playlists déjà générées par lui-même ou
selon ce principe.

Program that create or modify playlists expected to be read in a
random non-decremental way, in which case this generator allow to
add a probability to be played to each file, this being expressed
in the output playlist by a repeat of the entry for that file.
The program can read playlists already generated by itself or
following this method.
